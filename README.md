# Restaurant Group Multi-Location Operations Management Discord Bot

A Discord bot for multi-location restaurant groups and food service operators coordinating across venues. Posts daily sales summary reports by location, alerts general managers when revenue is below pacing, tracks employee scheduling conflicts across stores, manages inter-location inventory transfer requests, posts weekly food cost variance reports, and sends automated compliance reminders for health inspection dates and liquor license renewals.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
